# Nony.cc

> "Professional Next.js Hater."

My personal portfolio website built with Astro.

## Features

- **Astro v5**: Fast, content-focused.
- **View Transitions**: Smooth navigation animations.
- **Savage Menu**: Custom context menu that insults you for right-clicking.
- **Pixelated Profile**: Easter egg that clears up only if you agree that Next.js sucks (`?nextjs=sucks`).
- **Theme**: Dark/Light mode, "Glassmorphism" aesthetic.
- **Icons**: Tabler Icons.

## Tech Stack

- **Framework**: [Astro](https://astro.build)
- **Styling**: Vanilla CSS (Scoped)
- **Icons**: Tabler Icons (CDN)
- **Font**: Google Fonts (Fira Code, Fraunces, Outfit)

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Configuration

Edit `src/config.json` to update:
- Name / Bio
- Social Links
- Tech Stack (`/uses`)
- Savage Menu Messages
